# HR Analytics Portfolio – Arsh Mehra

This repository showcases my HR Analytics and Workforce Analytics projects built using Excel and Tableau. The focus is on transforming HR data into actionable insights to support data-driven decision-making across performance, attrition, and recruitment.

---

## 📊 Projects

### 1. Workforce Performance & Analytics Dashboard
- Built using Excel and Tableau
- Analyzes employee performance across tenure, training, job satisfaction, and overtime
- Identifies key drivers of workforce productivity and performance trends

**Key Insights:**
- Mid-tenure employees (3–5 years) show highest performance levels  
- Training has a positive impact on performance  
- Overtime does not consistently improve productivity  
- Job satisfaction correlates strongly with performance  

**Files:**
- Workforce Performance & Analytics Dashboard (.twbx)  
- Excel Dashboard (.xlsx)

### 2. Employee Attrition & Retention Analysis
- Built using Tableau
- Analyzes attrition trends across departments, job roles, and employee segments
- Identifies key factors influencing employee turnover

**Key Insights:**
- Higher attrition observed in specific roles and departments  
- Work experience and job satisfaction significantly impact retention  
- Certain employee segments show higher risk of leaving  

**Files:**

### 3. Recruitment Analytics Dashboard
- Built using Excel
- Tracks hiring funnel metrics and recruitment efficiency
- Helps analyze time-to-hire, candidate pipeline, and hiring performance

**Key Insights:**
- Identified bottlenecks in recruitment funnel  
- Improved visibility into hiring stages and conversion rates
