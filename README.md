# HR Analytics Portfolio – Arsh Mehra

This repository showcases my HR Analytics projects focused on workforce performance, recruitment, and employee insights using Excel and Tableau.

## 📊 Projects

### 1. Workforce Performance Dashboard
- Built using Excel and Tableau
- Analyzes employee performance across tenure, training, satisfaction, and overtime
- Provides insights into key drivers of productivity

## 🔍 Key Insights
- Mid-tenure employees (3–5 years) show highest performance levels
- Training has a positive impact on performance
- Overtime does not consistently improve productivity
- Job satisfaction correlates with performance

## 🛠 Tools Used
- Advanced Excel (Pivot Tables, Dashboards)
- Tableau (Data Visualization, Interactive Dashboards)
